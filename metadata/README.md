# Store Screenshots

Raycast renders PNG files in this folder on the extension's store page.

**Requirements**
- PNG, 2000 x 1250 pixels (16:10), ideally under 1 MB each.
- Show the current MiMo TTS commands only. Do not reuse screenshots from other extensions.

**Suggested captures for MiMo TTS**
1. `quick-read.png` — Quick Read with a selection and Toast "Reading …".
2. `tts-studio.png` — TTS Studio form with voice + opening style + rhythm tags selected.
3. `read-with-voice.png` — voice list with details panel visible for a recommended voice.
4. `select-voice.png` — Select Quick Read Voice list with the Current section pinned.
5. `preferences.png` — MiMo TTS preferences showing the `tp-` API Key hint and default voice.

Use Raycast's built-in screenshot tool (press `⌘⇧P` in a command, search "Take Screenshot for Store") to capture clean frames at the correct size.
